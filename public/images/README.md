## 推荐图片 (Unsplash)
位置: `/images/resources.json`

可以直接复制这些链接到 Cloudinary 或其他图床使用。

### 使用方法

1. **下载图片**: 点击上面的链接下载原图
2. **上传到 Cloudinary**: 在管理后台直接上传
3. **或直接使用链接**: 直接复制 URL 到产品图片字段

### 图片风格
所有图片均符合 NOREVA 品牌调性：
- 简约、优雅
- 黑白或中性色调
- 高质感
- 奢侈品风格

## 自定义图片建议

如果你想用自己拍摄的图片，建议：
- 尺寸: 1200x1600 (4:3 竖版) 或 800x800 (正方)
- 格式: WebP (自动转换)
- 风格: 简约白底或中性背景
- 光线: 自然光或柔和灯光
